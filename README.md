# PRogress

Application web (non finalisée) de **suivi de séances de musculation**, permettant de créer des programmes, enregistrer les performances, suivre l’historique des séances et visualiser l’évolution des performances via des graphiques.

Les données sont stockées localement dans le navigateur a l'aide de **localForage**.

## Fonctionnalités

- Création de programmes d’entraînement
- Ajout d’exercices avec séries, répétitions et poids
- Lancement et suivi d’une séance avec timer
- Marquage des exercices terminés
- Historique des séances effectuées
- Visualisation des performances (poids, séries, répétitions) sous forme de graphique
- Stockage local persistant (offline)

## Technologies utilisées

- **Preact**
- **Preact Router**
- **Chart.js**
- **localForage**
- **CSS Modules**
